# Start here
### Go to SWAT2: https://jazzauth.rchland.ibm.com:9443/SWAT2/requests/new/ to begin the onboarding process.

### Main Steps to Complete 
- Set the element
- Set the authorities
- Set the account
- Add a justification (optional)
- Submit request

- 'Authorities' is automatically set to 'Member'.
